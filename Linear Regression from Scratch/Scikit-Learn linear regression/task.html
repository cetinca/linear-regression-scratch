Description

You have built your own linear regression model and tested its performance. This stage will check how well your model performs against the LinearRegression algorithm from sklearn library.

Import the linear regression algorithm and metrics first. Note that sklearn has only MSE metric, so you have to take additional action to calculate RMSE. After that, initialize the algorithm as shown below:

from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error, r2_score

regSci = LinearRegression(fit_intercept=True)

Objectives

In this stage, your program should:

  1. Load data into pandas DataFrame;
  2. Import LinearRegression and metrics from sklearn;
  3. Fit the data by passing the \(X\) DataFrame and \(y\) Series to LinearRegression and CustomLinearRegression;
  4. Predict the \(\hat{y}\) values with LinearRegression and CustomLinearRegression;
  5. Calculate the RMSE and R2 metrics for the predictions of both implementations;
  6. Check how much the calculations of the two implementations differ: print the differences between intercept, coefficient, RMSE, and R2 values of LinearRegression and CustomLinearRegression as a Python dictionary.
